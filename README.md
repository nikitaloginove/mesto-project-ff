# Проектная работа Mesto


## В работе реализовано:

* _страница с отображением карточек_
* _возможность добавлять и удалять карточку_
* _возможность лайкать карточки и смотреть сколько всего лайков_
* _возможность изменять данные пользователя, в том числе аватар_
* _открывать карточки для более детального рассмотра изображений_
* _настроена валидация форм_
* _интерфейс подключен к API_
